# MaCache v4.0 开发计划

## 目标

在 v3.0 基础上，实现 AOF（Append Only File）持久化，服务器重启后数据不丢失。

当前状态：v3.0 已完成，支持五种数据类型 + TTL + 类型安全检查。

## 第一步：AOF 写入

每次执行写命令后，将原始 RESP 格式的命令追加写入 AOF 文件。

### 写入时机

在 `MaCacheHandler` 执行命令成功后，判断是否为写命令，若是则追加写入。

### 写命令列表

```
SET, DEL, INCR, DECR, MSET,
LPUSH, RPUSH, LPOP, RPOP,
SADD, SREM, SPOP,
HSET, HDEL,
ZADD, ZREM,
EXPIRE, PEXPIRE, PERSIST
```

### AOF 文件格式

与 RESP 协议完全一致，直接将客户端发来的原始消息追加写入：

```
*3\r\n$3\r\nSET\r\n$4\r\nname\r\n$7\r\nmacache\r\n
*2\r\n$4\r\nINCR\r\n$7\r\ncounter\r\n
```

### 核心类

```
persistence/
├── AofWriter.java     ← 负责追加写入 AOF 文件
└── AofLoader.java     ← 负责启动时重放 AOF 文件
```

**AofWriter.java**

```java
@Component
public class AofWriter {
    private static final String AOF_FILE = "macache.aof";
    private final BufferedWriter writer;

    public AofWriter() throws IOException {
        writer = new BufferedWriter(new FileWriter(AOF_FILE, true));
    }

    public synchronized void append(String respCommand) throws IOException {
        writer.write(respCommand);
        writer.flush();
    }
}
```

## 第二步：AOF 加载（重放）

服务器启动时，若 AOF 文件存在，逐条读取并重新执行命令，恢复内存状态。

**AofLoader.java**

```java
@Component
public class AofLoader {
    public void load(MaCache cache) throws IOException {
        File file = new File("macache.aof");
        if (!file.exists()) return;
        // 读取文件，按 RESP 格式解析每条命令
        // 调用 Commands.get(cmd).exec(cache, args) 重放
    }
}
```

重放时需要跳过 TTL 相关命令（EXPIRE/PEXPIRE），因为过期时间已经是绝对时间戳，重放后会立即过期。改为记录绝对过期时间戳的专用命令 `EXPIREAT`。

---

## 第三步：集成到启动流程

在 `MaCacheServer.startup()` 之前，通过 `MaPlugin` 的 `init()` 阶段加载 AOF：

```java
// MaCacheServer.java
@Override
public void init() {
    aofLoader.load(cache);  // 先恢复数据
    bossGroup = new NioEventLoopGroup(1);
    workerGroup = new NioEventLoopGroup(16);
}
```

## 第七步：验证方式

```bash
# 启动服务
mvn spring-boot:run

# 写入数据
redis-cli -p 6379 SET persistent_key hello_world
redis-cli -p 6379 LPUSH mylist a b c
redis-cli -p 6379 HSET user name kim age 30
redis-cli -p 6379 ZADD rank 100 alice 90 bob

# 查看 AOF 文件内容
cat macache.aof

# 重启服务器（Ctrl+C 后重新启动）
mvn spring-boot:run

# 验证数据恢复
redis-cli -p 6379 GET persistent_key
# 期望: "hello_world"

redis-cli -p 6379 LRANGE mylist 0 -1
# 期望: 1) "c"  2) "b"  3) "a"

redis-cli -p 6379 HGETALL user
# 期望: name, kim, age, 30

redis-cli -p 6379 ZSCORE rank alice
# 期望: "100.0"

# AOF 重写
redis-cli -p 6379 BGREWRITEAOF
# 期望: Background append only file rewriting started
```
